Project: Alexa Python Image Manipulation
Course: CSE (Computer Science and Principles of Engineering)
Duration: approximately 2 weeks (March 2018)
Objective: Create a brand image that represents their product (Alexa), in a memorable fashion, using Python. The purpose is to increase sales.
Type of Project: Python Image Manipulation - Partner Project

My partner for this project was Akamai Wong. The plan for the image was to compile logos that the brand uses (i.e. Dominos, Spotify, Phone, etc) and to use a function that pastes the images on the cropped image of the Alexa circle that we had. After, we had to modify images under one directory by resizing, cropping, and changing transparency values. We had a difficult time in creating loops for the code to be less file specific. We overcame this struggle using external resources as Google. We recorded our process in our Python Project Notebook. We displayed significant teamwork by dividing up parts of the project, so we would have equal amounts of work; if we had problems we would help each other out.

Project: Solar Hydrogen Vehicle
Course: POE (Principles of Engineering)
Duration: November 2018
Objective: Test hydrogen and solar cells to see which would be more compatible to power a vehicle.
Type of Project: Design + Build Project

For this project, we had to design a vehicle that could accodomate a hydrogen cell and a solar cell to power the vehicle, and provide a recommendation the company, Tesla. Tesla needs insight for how to use solar and hydrogen to power their cars. We are providing them with a prototype of a vehicle that will be able to use the two power sources in the vehicle that we built. We needed to create an efficient vehicle in which the power source can be easily switched between solar and hydrogen in order to determine the better source . The original design of the hydro-solar car consisted of three wheels, a simple gear train, and a platform for mounting the hydrogen and solar cells. Later, we made adjustments to the vehicle by making the sizing changes to be smaller and more compact in order to comply with the requirements. Our final recommendation for Tesla is to go with a solar-powered car. A solar-powered car is faster and more efficient overall. Documentation
